# General Track (Non-Technical)

For interns and contributors focusing on marketing, relations, community, operations, or other non-technical roles.

## Suggested Path

### Week 1: Understanding the Team

**Required Reading** (read these first, no need to create issues):
- [Playing on an Organic Team](https://www.whatscookin.us/playing-on-an-organic-team/)
- [How to Jump on a Moving Project](https://www.whatscookin.us/how-to-jump-on-a-moving-project/)
- [ROLES.md](https://github.com/Cooperation-org/coop-projects/blob/main/ROLES.md) in coop-projects repo

**Tasks** (create issues using these templates):
- [001] Create Team Calendar Invite
- [002] Make Claim on dev.linkedtrust.us
- [003] Attest to Teammate's Claim
- [008] Report in Standup

### Week 2: Applying Concepts

**Tasks:**
- [005] Leave Breadcrumbs (update docs with something you learned)
- [006] Seek Critique in Public (share work-in-progress)
- [007] Ask Questions Usefully
- [009] Review Teammate's Work

### Week 3: Real Work

**Tasks:**
- [004] Make Real Claim on linkedtrust.us (create a certificate-worthy claim)
- Find a real task in your area (marketing, relations, etc.) - ask your lead
- Document a process you followed
- Request attestations for work completed

## Completion

You're done when:
- All core tasks completed (001-009, excluding 101+)
- At least 2 peer reviews received
- At least 1 real deliverable for the team
- Published certificate on linkedtrust.us
